- ### Academic Performance — Analysis & Prediction

I analyzed a student dataset to discover the factors that most influence academic performance. I used exploratory data analysis techniques and machine learning models to predict student outcomes from demographic variables, identifying key patterns that can help improve educational decision-making.

- ### Global Music Streaming Analytics (Excel Dashboard)

I built an Excel dashboard using data from 50,000 music streaming sessions to explore genres, countries, and platforms, with KPIs for listening time, premium users, and song repetition. It includes interactive filters and a reproducible workflow with Power Query, pivot tables, and slicers.

![Dashboard demo](assets/img/proyectos/dashboard.gif)

**Technologies:** Microsoft Excel, Power Query, Pivot Tables, Slicers, Git, GitHub, Kaggle
[🔗 View repository on GitHub](https://github.com/gustavo21math/excel-dashboard)
- ### UPTC Editorial Database Scraper & Search Tool

I developed a tool that automates data collection from the UPTC Editorial using web scraping techniques. The system extracts relevant information from books and publications, stores the data in a NoSQL database (MongoDB), and lets users search and filter results through a simple interface. This project makes academic and bibliographic information more accessible to students and researchers.

![Search tool demo](assets/img/proyectos/demo.gif)

**Technologies:** Python, BeautifulSoup, MongoDB, Gradio, LangChain

[🔗 View repository on GitHub](https://github.com/gustavo21math/Editorial_Uptc/tree/main)
